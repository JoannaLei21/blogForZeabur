Project Diary 12/15

遇到困難

花太多時間在出設計圖 今天本來想要把 LINE 收款的功能串起來 不過組員提出，想一起把 RWD 切出來 或許是我自己莫名的堅持，決定還是出設計圖給組員製作 然後想到後面“我的頁面”也毀是很多功能的入口，應該也是要先設計出來 所以還是花大大部分的時間在出設計圖

自己也發覺，這樣花太多時間在設計上面，完全沒有碰到 code 真的是不太妙 所以跟組員協調，先把目前有的切出來！！！ 然後做頁面的功能， 之後往外連結的頁面，用到再說
